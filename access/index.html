Users API

Users API is a free online api for your shopping, e-commerce and login website. You can also use it for your programming course.Others use it for demo class, testing and etc.

Routes

These HTTP methods are supported

  • GET
  • /users
  • GET
  • /user/1
  • GET
  • /user?name=John
Resources

User API has one main resource, i.e, users for multiple website tasks.

  • Users

Example Codes

Using Fetch

Grab the below code and test it out

fetch('https://usersapi.vercel.app/users') .then(response => response.json()) .then(data => console.log(data))

Using Axios

Install axios and import it to your current working file before testing the below code. For simple html-css-js project you should just import axios cdn.

axios.get('https://usersapi.vercel.app/users') .then(response => response.json()) .then(data => console.log(data))
